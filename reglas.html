Regles del Torneig Pokémon

  1. Format de Combat

    Batallas 6vs6 con el sistema de combate estándar de Pokémon Escarlata/Púrpura.

  2. Sistema de draft

    Abans de cada combat entre dos entrenadors, cada un dels entrenadors podrà banejar un pokemon de l'altre persona. El primer enfrontament el entrenador qui baneji serà al atzar, per les següents el perdedor de l'anterior enfrontament decideix primer.

  3. Sistema de Puntuació

    Victoria = 3 punts, Empat = 1 punt, Derrota = 0 punts.

  4. Vidas Pokemon

    Si el pokemon mort durant l'aventura 3 cops, aquest es considera mort.

  5. Pokemon valids

    Nomes es poden fer servir pokemons que toquin pel sistema de nuzlocke.

  6. Pokemon shiny

    En cas de trobar-se un pokemon shiny aquest pot ser ficat al equip.

  7. Etapes

    El joc conta de 18 "medalles", cada 3 "medalles" es farà un combat entre tots els entrenadors.
